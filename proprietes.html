Les propriétés et les attributs css

Mise en forme du texte

font-style : la police : helvetica, arial, Times New Roman, Times, Comic
font-size : la taille :12px, 14px, 18px ...
font-weight : la graisse : bold
text-align : l'alignement du texte : left, center, right
text-decoration : la décoration du text (soulignement) : none, underline
color : la couleur du texte : red, #ff0000; (voir plus bas)

Mise en forme des cadres et des boites avec les balises div et span

width : la largeur en pixel ou en % : 220px, 35%
height : la hauteur en pixel ou en % : 440px, 70%
padding : la marge a l'intérieur de la boite : 10px
margin : la marge a l'extérieur de la boite :10px
backgroud-color : la couleur de fond : white, #ffffff
position : le positionnement absolu ou relatif du cadre : absolute, relative
align : l'alignement de la boite : left, top, right, bottom
border : La bordure : solid, groove black, red 2px,5px
float : Le coté vers lequel l'élément est aspiré : left, top, right, bottom
position : la maniére dont l'élément est positionné : relative, absolute

Les couleurs html/css

Déclaration des valeurs par leur nom: blue, red, yellow, black, white, orange, green...
Déclaration par les codes de couleurs RVB: code hexadécimale de #000000 a #ffffff (de noir a blanc)
bleu = #0000ff;
rouge = #ff0000;
vert = #00ff00
